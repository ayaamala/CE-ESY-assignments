# CE-ESY-assignments
في البداية هيكل البيانات (RingBuffer)يضم مصفوفة البيانات (data_store) ومؤشرات التحكم بموقع القراءة (read_pos) وموقع الكتابة (write_pos) بالإضافة إلى عداد العناصر النشطة (active_elements)
 اما دالة الإعداد(setupBuffer) تعمل على تهيئة المخزن وتصفير كافة المؤشرات والعدادات عند بدء التشغيل.
دالة الإضافة (pushData): المسؤولة عن اضافة البيانات في المخزن. تتضمن نظام حماية يمنع الكتابة في حال امتلاء المخزن (Buffer Overflow)
ودالة السحب (popData): المسؤولة عن ازالة البيانات بالترتيب الصحيح تتضمن نظام حماية يمنع القراءة من مخزن فارغ (Buffer Underflow)

📌استقبال المدخلات: يأخذ البرنامج اسماً من المستخدم ويضيف إليه اللاحقة الخاصة بالمقرر CE-ESY
📌عملية التخزين: يتم تمرير السلسلة النصية حرفاً بحرف إلى دالة pushData ليتم ترتيبها دائرياً داخل الذاكرة
📌عملية العرض: يتم سحب الأحرف تباعاً عبر دالة popData وطباعتها حتى يفرغ المخزن تماماً
📌في النهاية للتحقق يقوم البرنامج بتقديم تأكيد مرئي بأن النظام أصبح فارغاً وجاهزاً للاستخدام مرة أخرى (System is clear)
